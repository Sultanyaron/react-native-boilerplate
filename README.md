## Usage

### Clone

- Clone this repo to your required directory

```shell
$ git clone https://github.com/Sultanyaron/react-native-boilerplate.git
```

### Install Dependencies

```shell
$ yarn install
```

### Update pods

```shell
$ cd ios
$ pod install
$ cd ..
```

### Run the project

```shell
$ yarn run ios
$ yarn run android
```
